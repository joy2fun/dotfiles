Usage
========

- setup login
- setup git
- setup zsh
- clone oh-my-zsh
- clone dotfiles
- setup vimfiles
- setup powerline

## Windows MINGW64

```sh
git clone https://github.com/joy2fun/dotfiles.git
ln -s ~/dotfiles/bashrc ~/.bashrc
ln -s ~/dotfiles/.gitignore_global ~/.gitignore_global
ln -s ~/dotfiles/.gitconfig-win ~/.gitconfig
```
